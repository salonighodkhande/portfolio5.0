Day 1: HTML Basics

  • Understanding the basic HTML structure: <html>, <head>, <body>.
  • Learning headings <h1> to <h6>.
  • Paragraphs <p>, line breaks <br>, and text formatting tags.
  • Lists: ordered <ol> and unordered <ul>.
  • Adding comments <!-- comment --> for code readability.

Day 2: HTML Tags & Attributes

  • Hyperlinks using <a> and href.
  • Images <img> with src and alt.
  • Tables using <table>, <tr>, <td>.
  • Forms: <form>, <input>, <button>.
  • Using id and class for styling.

Day 3: CSS Basics

  • Ways to include CSS: inline, internal, external.
  • Selectors: element, class ., id #.
  • Properties: color, background-color, font-size.
  • Margin, padding, borders, and box model basics.
  • Styling headings, paragraphs, and buttons.

Day 4: Advanced CSS

  • Box model in depth: content, padding, border, margin.
  • Flexbox layout for horizontal and vertical alignment.
  • Hover effects and transitions.
  • Gradients, shadows, and border-radius for modern design.
  • Combining multiple CSS properties for cards and sections.

Day 5: Responsive Design

  • Importance of responsive design.
  • Media queries for mobile, tablet, desktop layouts.
  • Relative units: %, em, rem.
  • Responsive navigation and content sections.
  • Testing on different screen sizes.

Day 6: JavaScript Basics

  • Variables: let, const, var.
  • Data types: string, number, boolean, array, object.
  • Operators: arithmetic, comparison, logical.
  • Functions: defining and calling simple functions.
  • DOM manipulation basics: document.getElementById(), querySelector().
