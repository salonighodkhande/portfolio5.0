Day 1: Introduction to Networking

The session covered the basics of data communication and networking. Students understood how devices communicate and why networking is crucial.

  • What is Data Communication and Networking.
  • Importance of data communication and networking.
  • Components of data communication: sender, receiver, medium, protocols.
  • Characteristics of data communication: speed, accuracy, reliability.
  • Types of communication: simplex, half-duplex, full-duplex.
  • Types of data transmission: analog, digital.

Day 2: Network and Data Types

Explored network types, definitions, and data classifications that travel across networks.

  • Definition of a Network.
  • Types of networks: LAN, WAN, MAN, PAN.
  • Networking concepts and their importance.
  • Types of data: oriented, observed, inferred.

Day 3: OSI Model

The OSI model explains how data moves through seven layers, with each layer performing specific functions for successful communication.

  • 7 layers of OSI Model: Physical, Data Link, Network, Transport, Session, Presentation, Application.

Day 4: HTTP Methods & Domain Name System

This session covered HTTP methods for web communication and the DNS system for resolving addresses.

  • HTTP Methods: GET, POST, PUT, DELETE.
  • Domain Name System (DNS) and its role.
  • How browsers use HTTP and DNS to load web pages.

Day 5: Test on OSI Model and DHCP

Students were tested on OSI model concepts and DHCP was introduced for dynamic IP assignment.

  • Review of OSI Model layers.
  • Introduction to DHCP (Dynamic Host Configuration Protocol).
  • Understanding dynamic IP address assignment.

Day 6: DHCP Acknowledge, ARP & ICMP

Covered DHCP acknowledgements, ARP protocol for IP-MAC mapping, and ICMP for diagnostics and error messages.

  • DHCP Acknowledge: Server confirms IP assignment.
  • ARP: Maps IP addresses to MAC addresses.
  • ICMP: Sends network error messages and diagnostics (e.g., ping).
