Day 1 – Introduction to AI & ML

We started the AI & ML module by understanding the fundamental concepts, including definitions, history, and importance of Artificial Intelligence and Machine Learning in modern technology. Topics included supervised vs unsupervised learning, real-world applications, and a brief overview of Python as the primary language for ML.

  • Understanding AI & ML concepts
  • History and evolution of AI
  • Applications in healthcare, finance, and robotics
  • Introduction to Python for ML

Day 2 – Fundamentals of Python

We focused on Python fundamentals crucial for AI & ML, covering variables, data types, conditional statements, loops, functions, and object-oriented programming basics. We practiced writing small programs to understand Python syntax, and emphasized code readability and debugging techniques.

  • Variables, data types, and operators
  • Conditional statements and loops
  • Functions and modular programming
  • Introduction to object-oriented programming
  • Debugging and code best practices

Day 3 – Python Libraries: NumPy Part 1

We started exploring NumPy, a library for numerical computing in Python. Topics included arrays, array operations, indexing, slicing, and basic mathematical functions. Hands-on practice included creating arrays, performing arithmetic operations, and understanding matrix manipulations.

  • Introduction to NumPy
  • Creating 1D and 2D arrays
  • Array indexing and slicing
  • Mathematical operations on arrays
  • Matrix manipulations and shapes

Day 4 – NumPy Part 2 & Matplotlib Introduction

We continued with NumPy focusing on advanced operations like broadcasting, aggregation, and vectorization. We also started Matplotlib, learning basic plotting functions, line plots, and customizing axes, labels, and titles for data visualization.

  • Broadcasting and vectorized operations
  • Aggregations: sum, mean, min, max
  • Introduction to Matplotlib
  • Line plots, axis customization, and titles
  • Creating multiple plots and legends

Day 5 – Matplotlib & Seaborn Introduction

We continued learning Matplotlib with bar charts, scatter plots, histograms, and customization techniques. Later, we started Seaborn to create more aesthetically appealing statistical plots, including distribution plots, box plots, and pair plots.

  • Bar charts, histograms, scatter plots
  • Plot customization: colors, labels, grid
  • Introduction to Seaborn library
  • Distribution plots, box plots, and pair plots
  • Understanding visualization best practices

Day 6 – Seaborn Continued

We focused on Seaborn advanced plots like heatmaps, regression plots, and categorical plots. Learned how to combine Matplotlib and Seaborn for powerful visualizations and started working on mini-projects using sample datasets.

  • Advanced Seaborn plots: heatmaps, regression plots
  • Categorical plots: countplot, boxplot, violinplot
  • Combining Matplotlib & Seaborn
  • Mini-projects with real datasets

Day 7 – Seaborn Projects & Recap

We continued with Seaborn for advanced data visualization projects. The session included consolidating all Python skills, creating final plots for datasets, interpreting insights, and preparing for presentations.

  • Advanced visualizations with Seaborn
  • Project work with real datasets
  • Analyzing insights from plots
  • Final recap of Python, NumPy, Matplotlib, Seaborn
