Module 2 - Computer Aided Designing

Home

Day 1: Introduction to CAD/Basic Modeling

  • What is CAD and its importance in engineering and design.
  • Types of CAD: 2D CAD, 3D CAD.
  • Popular CAD software: AutoCAD, SolidWorks, Fusion 360.
  • Understanding the CAD workspace: menus, toolbars, drawing area.
  • Basic drawing commands: Line, Circle, Rectangle, Arc.

Day 2: Sketching and Basic Features

  • Creating and managing sketches.
  • Using constraints to define sketch geometry.
  • Basic features: Extrude, Revolve, Sweep, Loft.
  • Modifying features: Fillet, Chamfer, Shell.
  • Understanding the feature tree/history.

Day 3: Dimensions & Annotations

  • Adding dimensions: linear, angular, radius.
  • Using text annotations to label parts.
  • Leaders and hatching for clarity.
  • Maintaining drawing standards: units, scale.

Day 4: Advanced 2D CAD Techniques

  • Polylines, splines, and complex curves.
  • Trim, extend, offset, fillet, chamfer commands.
  • Creating blocks for reusable components.
  • Working with layouts and paper space.

Day 5: Introduction to 3D CAD

  • Difference between 2D and 3D CAD.
  • 3D workspace, views, and navigation.
  • Basic 3D objects: Extrude, Revolve, Loft, Sweep.
  • Understanding coordinates, axes, and planes in 3D.

Day 6: 3D Modeling Techniques

  • Modifying 3D models: Move, Rotate, Scale.
  • Combining 3D objects: Union, Subtract, Intersect.
  • Using Fillet and Chamfer in 3D.
  • Introduction to assemblies: assembling multiple components.

Day 7: Rendering & Output

  • Applying materials, colors, and textures.
  • Creating realistic renders and views.
  • Generating technical drawings from 3D models.
  • Exporting for manufacturing, 3D printing, or documentation.
  • Tips for professional CAD documentation and presentation.
